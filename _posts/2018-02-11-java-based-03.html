2.修饰符default

证明:当一个类没有写访问控制修饰符(即default)：

01.该类可以被本包的类继承

02.该类不可以被其他包的类继承

03.一个.java文件中可以有无数个default修饰的类


下面只给出证明同个包下的代码(不同包请自行测试):

{% highlight ruby %} class Test { } class Test02 extend Test { } {% endhighlight %}

3.修饰符为public

证明: 当一个类的问控制修饰符为public:

01.该类可以被本包的类继承

02.该类可以被其他包的类继承

03.该类必须与所在的.java文件名称一致。

04.一个.java文件中可以有0个或1个public修饰的类


下面只给出证明同个包下的代码(不同包请自行测试):

{% highlight ruby %} public class Test { } public class Test02 { //编译器错误提示：The public type Test02 must be defined in its own file } class Test03 extend Test { } {% endhighlight %}

4.总结

修饰类的时候(内部类除外)，还是比较好记的:

1.不能使用private或者protected

2.一个.java文件夹中可以包含0或1个public修饰的类

3.一个.java文件夹中可以包含0或多个default修饰的类

4.public修饰的类，可被本包或其他包的类继承

5.default修饰的类，可被本包的类继承,不可以被其他包的类继承

6.一旦使用public修饰类,该类必须和.java文件名称一致
